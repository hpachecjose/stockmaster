### Exemplos de Aplicações em COBOL
Em ambientes de grande porte, as aplicações de COBOL são geralmente estruturadas para lidar com o ciclo completo de dados, desde a entrada de mercadorias até o faturamento. [youtube](https://www.youtube.com/watch?v=7xgTjbJDgxs)

- **Controle de Estoque e Compras:** Programas COBOL realizam o controle rigoroso de entradas e saídas (baixas) de produtos, integrando lógica de validação para impedir retiradas superiores ao saldo disponível e automatizando alertas para estoque mínimo. [youtube](https://www.youtube.com/watch?v=gc9OF1EWPGc)

- **Cadeia de Suprimentos:** O COBOL processa algoritmos de gestão de suprimentos que definem lotes de compra e intervalos de pedido para categorias de itens (ex: itens de alto ou baixo custo), otimizando o nível de estoque conforme a necessidade da operação. 6. Fluxo de Autenticação
Usuário digita e-mail + senha
        │
        ▼
API recebe → valida formato
        │
        ▼
Verifica se conta está bloqueada (Redis)
        │
        ├── Bloqueada → retorna erro + tempo restante
        │
        ▼
Busca usuário no PostgreSQL
        │
        ▼
Verifica hash Argon2id
        │
        ├── Falhou → incrementa tentativas (máx 5) → após 5: bloqueia 30min
        │
        ▼
Gera JWT (15min) + Refresh Token (7 dias) → salva no Redis
        │
        ▼
Retorna tokens ao frontend
Recuperação de senha:

Usuário informa e-mail
    → API gera token aleatório de 32 bytes
    → Salva hash do token no banco (válido 30min)
    → Envia link por SMTP: https://app/reset?token=XXX
    → Usuário clica → valida token → redefine senha → token invalidado
